ex> 회원가입 -> 가입이 완료되었습니다. -> 확인 버튼 누르면 로그인 창으로 이동

- [x] 완료
  → `FeedbackProvider`(Modal 기반 공통 안내창, 웹에서 안 되는 Alert 대체) + `useFeedback().show({...onConfirm})`.
  → 회원가입 완료→확인→로그인 이동, 그룹 생성/참여 성공·실패 안내에 적용.

### 3. 하단 탭 UI
- 홈, 일정, 가계부, 유틸, 전체보기 로 구성
- 전체 보기를 누르면 현재 유틸 메뉴와 똑같은 구성으로 위시리스트, 제태크 표시
- 하단 탭 UI는 상단 - 아이콘, 하단 - 텍스트로 배치
- 홈은 아이콘만 놓고, 텍스트 제외

- [x] 완료
  → 탭을 홈/일정/가계부/유틸/전체보기 5개로 재구성(`(tabs)/_layout.tsx`), 위시·재테크는 `href:null`로 숨김.
  → 홈은 `tabBarLabel:()=>null`로 아이콘만. 전체보기(`(tabs)/more.tsx`) 신규 — 위시리스트/재테크/유틸 항목 메뉴.

### 4. 메인 페이지 위젯

1. 가족 멤버 위젯 
1) 화면 구성

2. 일정 위젯
- 금일 등록된 일정 출력

- 사용처 도넛 그래프 표시
