### 1. 화면 UI 변경
- IOS / ANDROID / WEB 전체 적용될 수 있는 UI (* KAKAO BANK APP UI 참고)
- Phosphor Icons 활용
- 모든 페이지에 padding 적용 (적당한 여백)
ex> main 페이지에 그룹만들기, 초대코드로 참여 등 버튼이 padding 없이 width 100% 로 출력

- [🔁] 진행 중 (공통 기반 + 핵심 화면 완료)
  → 공통 컴포넌트 `Screen`(패딩 래퍼)/`Button`/`ScreenHeader` 작성, `phosphor-react-native` 도입.
  → 메인·탭바·그룹·인증 화면에 적용(버튼 여백/아이콘 그리드 정리). 나머지 세부 화면은 다음 단계.

### 2. 안내 메시지 창
- 사용자의 action이 완료됐을 때, 안내 메시지 창 및 redirect 설정
ex> 회원가입 -> 가입이 완료되었습니다. -> 확인 버튼 누르면 로그인 창으로 이동

- [✅] 완료
  → `FeedbackProvider`(Modal 기반 공통 안내창, 웹에서 안 되는 Alert 대체) + `useFeedback().show({...onConfirm})`.
  → 회원가입 완료→확인→로그인 이동, 그룹 생성/참여 성공·실패 안내에 적용.

### 3. 하단 탭 UI
- 홈, 일정, 가계부, 유틸, 전체보기 로 구성
- 전체 보기를 누르면 현재 유틸 메뉴와 똑같은 구성으로 위시리스트, 제태크 표시
- 하단 탭 UI는 상단 - 아이콘, 하단 - 텍스트로 배치
- 홈은 아이콘만 놓고, 텍스트 제외

- [✅] 완료
  → 탭을 홈/일정/가계부/유틸/전체보기 5개로 재구성(`(tabs)/_layout.tsx`), 위시·재테크는 `href:null`로 숨김.
  → 홈은 `tabBarLabel:()=>null`로 아이콘만. 전체보기(`(tabs)/more.tsx`) 신규 — 위시리스트/재테크/유틸 항목 메뉴.
